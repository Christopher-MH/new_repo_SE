# new_repo_SE
Repository untuk tugas Software Engineering

1. Judul
  Final Project, Semester 2, Data Structures

2. Deskripsi
  Program ini bertujuan untuk membantu manager investasi dalam menangani pencatatan portfolio saham klien mereka, dengan memanfaatkan BST (binary search tree). Program ini dapat bermanfaat untuk manager portfolio yang memiliki ratusan portfolio client yang harus diperhatikan. Dengan bantuan BST, program ini menjamin kecepatan dalam searching, inserting, deleting, dan viewing dari data customer dan portfolio mereka.

3. Cara menjalankan project
  Project dapat dijalankan langsung pada DevC++ atau aplikasi lain yang dapat menjalankan bahasa C. Project ini sudah ada UI dalam bentuk CLI untuk membantu navigasi segala functionality dari program.
